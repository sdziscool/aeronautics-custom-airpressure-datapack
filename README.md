These are datapacks, move the one you want to your worlds datapack folder and perform a '/reload' or just re-open your world.
ONLY INSTALL ONE OF THEM.
The name of the datapack is the height where the airpressure is 0, so the maximum height you're ever going to reach basically.
you can replace or remove them anytime you want to your liking.
I am not responsible for worlds crashing etc.

The datapack is essentially just a json file with values in it, you can adjust it to your liking if you can figure out how to properly calculate the slope (it's not magic, just a linear relationship).

good flying!

Quick install:
- download via green button as zip
- unzip wherever you want
- move whichever maximum flying height you want to your world datapack folder (google if you don't know where that folder might be
- '/reload' or just re-open your world, note that some flying machines might suddenly fly up real high!
